## 1. 产品概述

一款复古像素风格的 1v1 机甲对战小游戏，两名玩家在同一台电脑上进行实时对战。游戏包含完整的战斗逻辑、血量系统和胜负判定机制，形成可玩的 Demo 版本。

- 主要目标：提供双人本地对战娱乐体验，展示像素风格机甲战斗的核心玩法
- 目标用户：喜欢复古像素游戏、格斗游戏的休闲玩家
- 产品价值：轻量、即开即玩，无需联网，适合朋友间快速对战

## 2. 核心功能

### 2.1 玩家角色
| 角色 | 控制方式 | 核心能力 |
|------|----------|----------|
| 玩家1（红色机甲） | WASD + 空格键 | 移动、攻击、防御 |
| 玩家2（蓝色机甲） | 方向键 + Enter | 移动、攻击、防御 |

### 2.2 功能模块
1. **游戏主菜单**：标题画面、开始游戏按钮、操作说明
2. **对战场景**：像素风格战斗场地、双方机甲角色、实时 HUD
3. **战斗系统**：移动、攻击（近战）、防御、受击反馈、血量管理
4. **胜负判定**：血量归零时判定胜负，显示结果画面
5. **重新开始**：胜负画面后返回主菜单或重新对战

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主菜单 | 标题展示 | 像素风格游戏标题 "MECHA BATTLE" |
| 主菜单 | 操作按钮 | "开始游戏" 按钮 |
| 主菜单 | 操作说明 | 显示双方按键操作说明 |
| 对战场景 | 战斗场地 | 像素风格的地面场景，带网格或平台元素 |
| 对战场景 | 机甲角色 | 红蓝两台像素机甲，带简单动画 |
| 对战场景 | HUD | 双方血量条、角色名称 |
| 对战场景 | 战斗逻辑 | 攻击命中判定、防御减伤、受击后退 |
| 结果画面 | 胜负展示 | 显示获胜方，返回主菜单按钮 |

## 4. 用户界面设计

### 4.1 设计风格
- **风格定位**：复古 8-bit 像素风，NES/FC 时代的美学
- **主色调**：
  - 背景：深蓝紫色 (#1a1a2e)
  - 地面：灰褐色 (#8B7355)
  - 玩家1（红色机甲）：红色系 (#e74c3c, #c0392b)
  - 玩家2（蓝色机甲）：蓝色系 (#3498db, #2980b9)
  - 血量条：绿色 (#2ecc71) → 黄色 (#f39c12) → 红色 (#e74c3c)
- **字体**：Press Start 2P（像素风格字体，Google Fonts）
- **按钮样式**：像素块状按钮，带 hover 效果
- **布局风格**：居中布局，战斗场景居中，HUD 在顶部

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主菜单 | 标题区域 | 像素字体大标题，带闪烁动画 |
| 主菜单 | 按钮区域 | 像素风格按钮，hover 时变色 |
| 主菜单 | 说明区域 | 按键操作指引文字 |
| 对战场景 | 战斗区域 | Canvas 绘制的像素场景和角色 |
| 对战场景 | HUD区域 | 顶部左右两侧血量条 + 名字 |
| 结果画面 | 结果展示 | 半透明遮罩 + 获胜文字 + 重开按钮 |

### 4.3 响应式
- 桌面优先，固定游戏画布尺寸（800x500）
- Canvas 内自适应缩放

## 5. 游戏玩法设计

### 5.1 操作方式
| 操作 | 玩家1 | 玩家2 |
|------|-------|-------|
| 向左移动 | A | ← |
| 向右移动 | D | → |
| 跳跃 | W | ↑ |
| 攻击 | 空格键 | Enter |
| 防御 | S | ↓ |

### 5.2 战斗机制
- **血量**：每个机甲初始 100 HP
- **攻击**：近战攻击，命中造成 15 点伤害，击退对方
- **防御**：减少 60% 伤害（受到 6 点伤害），防御时不能移动
- **攻击冷却**：每次攻击后有 0.5 秒冷却时间
- **跳跃**：跳跃高度固定，跳跃时不能攻击/防御
- **受击后摇**：被击中后有 0.3 秒硬直时间
- **胜负**：一方血量归零时游戏结束，另一方获胜

### 5.3 场景设计
- 地面为像素风格的平台（深色地面线 + 浅色网格点缀）
- 场景两侧有边界墙
- 背景有简单的像素云朵或星空装饰

## 6. 动画与反馈
- **机甲待机**：轻微上下浮动
- **移动**：左右移动动画（步伐交替）
- **攻击**：手臂/武器向前伸出，伴随像素特效
- **防御**：机甲前出现护盾光效
- **受击**：闪烁 + 后仰/后退
- **血量变化**：血量条平滑减少
- **胜负**：结果文字渐入动画
