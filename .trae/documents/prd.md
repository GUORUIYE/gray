## 1. 产品概述

一款双人对战的像素风格机甲对战小游戏，玩家控制各自的机甲角色在战场上移动、攻击、防御，通过策略与操作击败对手。
- 游戏采用复古像素画风，包含基础场景（战场地面、背景元素）和角色动画（移动、攻击、受击、待机等）
- 支持双人键盘操作，实现即时对战体验

## 2. 核心功能

### 2.1 游戏模式
| 模式 | 说明 |
|------|------|
| 双人对战 | 两名玩家在同一台电脑上使用键盘进行对战（PvP） |

### 2.2 功能模块
1. **机甲控制模块**：移动（左右方向）、跳跃、攻击（近战/远程）、防御
2. **战斗系统模块**：攻击判定、防御减伤、伤害计算
3. **血量系统模块**：血条显示、伤害反馈、胜负判定
4. **场景渲染模块**：像素风格战场、背景滚动、地面平台
5. **动画系统模块**：待机、移动、攻击、受击、防御等角色动画

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主界面 | 标题画面 | 显示游戏标题，按任意键开始 |
| 游戏主界面 | 对战场景 | 双人机甲实时对战 |
| 游戏主界面 | 结算画面 | 显示胜负结果，按任意键重新开始 |

## 3. 核心流程

```
- 玩家进入游戏 → 标题画面
- 按任意键 → 进入对战
- 双人操控机甲对战 → 一方血量归零
- 显示胜负结果 → 按任意键重新开始
```

## 4. 用户界面设计

### 4.1 设计风格
- **美术风格**：复古像素风（8-bit/16-bit），使用块状像素角色和场景
- **主色调**：深蓝紫色夜空背景，暖色（橙色/红色）机甲 vs 冷色（蓝色/青色）机甲
- **UI风格**：像素字体，简单直接的 HUD 显示
- **字体**：使用像素风格字体（如 Press Start 2P 或自定义像素字体）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 标题画面 | 标题区 | 像素风格游戏标题 "机甲对决" |
| 标题画面 | 操作提示 | 显示双方按键说明 |
| 对战场景 | 战场 | 像素风格地面、背景建筑/天空 |
| 对战场景 | 机甲 | 像素风格机甲角色（含动画帧） |
| 对战场景 | HUD | 双方血条、角色名称 |
| 结算画面 | 结果展示 | 胜利文字、重新开始提示 |

### 4.3 响应式适配
- 专注于桌面端体验，固定分辨率画布
- Canvas 自适应缩放以适应窗口大小

## 5. 游戏玩法设计

### 5.1 操作方式
| 操作 | 玩家1（红色机甲） | 玩家2（蓝色机甲） |
|------|------------------|------------------|
| 向左移动 | A | ← |
| 向右移动 | D | → |
| 跳跃 | W | ↑ |
| 攻击 | S | ↓ |
| 防御 | Q | /（斜杠） |

### 5.2 战斗机制
- **攻击**：每次攻击造成 10 点伤害
- **防御**：防御状态下受到的伤害减半（5 点）
- **攻击冷却**：每次攻击后有 0.5 秒冷却时间
- **跳跃**：跳跃时无法攻击或防御
- **血量**：每台机甲初始 100 HP

### 5.3 胜负条件
- 一方 HP 降至 0 时，另一方获胜
- 显示胜利画面后，按任意键重新开始
